## Requirements

### Purpose

Publish a consumable v0.7.0 release for Rust and Go consumers after the
reader-window verifier repair and Python/Node parity reconciliation, so
Netdata integration can depend on a tagged SDK instead of a moving branch.

### User Request

Release `0.7.0` for Netdata integration.

### Assistant Understanding

- Rust workspace package version is currently `0.6.4`.
- Go module path is `github.com/netdata/systemd-journal-sdk/go` and requires
  `go 1.26`.
- Release tags `v0.7.0` and `go/v0.7.0` do not exist locally or remotely at
  activation time.
