# SOW-0117 - v0.7.2 Release

## Status

Status: in-progress

Sub-state: preparing release `0.7.2` after SOW-0116 retired Python and Node.js from product scope.

## Requirements

### Purpose

Publish a consumable Rust/Go `0.7.2` release that includes SOW-0116's product-scope cleanup, so Netdata integration can target a released SDK version whose active product contract is Rust and Go only.

### User Request

The user clarified that the next release should be `0.7.2`, because `0.7.1` was already released.

### Assistant Understanding

Facts:

- `v0.7.1` and `go/v0.7.1` already exist locally and remotely and peel to release commit `1df0eae1a31f80459453e0ca6a696282372b609c`.
- SOW-0114 records Rust crates published to crates.io at `0.7.1` and Go module lookup success for `github.com/netdata/systemd-journal-sdk/go@v0.7.1`.
- Current `master` is ahead of `origin/master` by SOW-0116 commit `f77975e`, which retired Python/Node product targets.
- The Go module lives under `go/`, so the release needs both `v0.7.2` and `go/v0.7.2` tags pointing to the same release commit.

Inferences:

- Reusing or moving `0.7.1` would be unsafe because the tags and crates.io packages are already public.
- `0.7.2` is the correct patch release for the post-SOW-0116 Rust/Go product scope.

Unknowns:

- Whether crates.io accepts all `0.7.2` package publishes without index propagation delays between dependent packages.

### Acceptance Criteria

- Rust workspace/package versions and public install snippets are updated from `0.7.1` to `0.7.2`.
- Rust publish dry-runs pass in dependency order, with dependency-order publish sequencing if crates.io index propagation requires it.
- Rust crates are published to crates.io at `0.7.2`.
- `master` is pushed with the release commit and SOW close commit.
- Annotated tags `v0.7.2` and `go/v0.7.2` exist locally and remotely and peel to the same release commit.
- `go list -m github.com/netdata/systemd-journal-sdk/go@v0.7.2` resolves after tag push.
- SOW audit, `git diff --check`, Rust/Go validation, and release-tag checks pass.
